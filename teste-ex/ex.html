Função print

A função print() em Python é uma maneira de mostrar informações na tela do usuário. É como se você estivesse escrevendo mensagens visíveis para si mesmo ou para os usuários do seu programa. Essas mensagens podem ser textos, números ou qualquer coisa que você queira que alguém veja.

Print e variáveis

Outra maneira de utilizar o comando print é com variáveis. Quando você usa o print com variáveis, você está exibindo o valor armazenado nessas variáveis. Por exemplo, se você tem uma variável chamada 'idade' com o valor 25, ao usar print(idade), o programa mostrará 25 na tela. É uma maneira de ver os dados que você armazenou nas variáveis durante a execução do programa.
